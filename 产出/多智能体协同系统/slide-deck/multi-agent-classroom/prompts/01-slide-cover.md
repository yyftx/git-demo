## STYLE_INSTRUCTIONS

Design Aesthetic: Clean, structured visual metaphors using blueprints, diagrams, and schematics. Precise grid-based layouts with engineering precision. Technical and analytical presentation with restrained, unified color treatment.

Background:
  Texture: Subtle grid overlay on light engineering paper feel
  Base Color: Blueprint Off-White (#FAF8F5)

Typography:
  Headlines: Bold geometric sans-serif, clean letterforms, authoritative technical presence
  Body: Clean serif body text, readable at smaller sizes, professional editorial quality

Color Palette:
  Background: Blueprint Paper (#FAF8F5) - Primary background
  Grid: Light Gray (#E5E5E5) - Background grid lines
  Primary Text: Deep Slate (#334155) - Headlines, body text
  Primary Accent: Engineering Blue (#2563EB) - Key elements, highlights
  Secondary Accent: Navy Blue (#1E3A5F) - Supporting elements
  Tertiary: Light Blue (#BFDBFE) - Backgrounds, fills
  Warning: Amber (#F59E0B) - Emphasis points

Visual Elements:
  - Precise lines with consistent stroke weights (1-2px)
  - Technical schematics and clean vector graphics
  - Straight lines or 90-degree angles only for connections
  - Grid alignment for all elements

Density Guidelines:
  - Content per slide: 3-5 key points maximum
  - Whitespace: 40-50% of canvas
  - Element count: 1-2 main visual elements per slide

Style Rules:
  Do: Maintain consistent line weights, use grid alignment, keep palette restrained, create clear hierarchy through scale
  Don't: Use hand-drawn shapes, add decorative flourishes, use curved lines, include photographic elements
